# 01 — 调度器与主循环

> 文件: `ArduPlane/Plane.cpp`
> 核心: scheduler_tasks[] 定义了所有任务的执行顺序和频率

---

## 调度任务表

scheduler_tasks[] 在 `Plane.cpp` 中定义，按优先级排列:

### FAST_TASK (每圈都跑)

| 顺序 | 函数 | 作用 |
|------|------|------|
| 1 | `ahrs_update()` | AHRS 姿态更新 |
| 2 | `update_control_mode()` | 检查模式切换 + 调用 `control_mode->update()` |
| 3 | `stabilize()` | 调用 `control_mode->run()` → PID 稳定 |
| 4 | `set_servos()` | 舵机/电机 PWM 输出 |

### 固定频率任务

| 频率 | 函数 | 作用 |
|------|------|------|
| 50Hz | `read_radio()` | RC 输入读取 |
| 50Hz | `read_rangefinder()` | 测距仪 |
| 50Hz | `check_short_rc_failsafe()` | 短时失控检测 |
| 10Hz | `navigate()` | 导航更新 |
| 10Hz | `update_alt()` | 高度控制 → TECS |
| 10Hz | `Log_Write_Fast()` | 快速日志 |
| 5Hz | `update_is_flying_5Hz()` | 飞行状态检测 |
| 5Hz | `crash_detection_update()` | 坠毁检测 |
| 3Hz | `check_long_failsafe()` | 长时失控检测 |
| 3Hz | `gcs_send_heartbeat()` | 地面站心跳 |
| 1Hz | `one_second_loop()` | 通道映射/更新 home/ADSB/罗盘 |

---

## 每圈控制流

```
read_radio()            ← 50Hz, 但不阻塞，有数据就读
    ↓
update_control_mode()   ← 检查 set_mode 请求 → 调用 control_mode->update()
    ↓
stabilize()             ← 调用 control_mode->run() → 姿态 PID
    ↓
set_servos()            ← 混控输出 → PWM

10Hz 补充路径:
  navigate() → update_alt() → TECS 计算高度/空速需求
                            → L1 计算横向导航需求
```

## 关键函数详解

### ahrs_update()

```cpp
// 每圈执行
void Plane::ahrs_update()
{
    // 1. 更新 AHRS (内部融合 IMU/GPS/Baro/Compass)
    ahrs.update();

    // 2. 计算当前姿态下的滚转/俯仰限幅
    // 3. 累积偏航误差 (用于地面转向)
    // 4. 更新 QuadPlane 惯性导航 (如果有)
}
```

### update_control_mode()

```cpp
void Plane::update_control_mode()
{
    // 1. 清除过期的 hold_course
    // 2. 刷新油门限幅 (THR_MIN/MAX)
    // 3. 调用当前模式的 update()
    control_mode->update();
    // 4. 检查 autoland 起飞方向
}
```

### stabilize()

```cpp
void Plane::stabilize()
{
    // 1. 如果启用了脚本控制 → 走脚本路径
    // 2. 否则调用当前模式的 run()
    control_mode->run();
    // 3. 地面零油门时重置积分器
}
```

### one_second_loop()

```cpp
void Plane::one_second_loop()  // 1Hz
{
    // - 重新映射控制通道 (RC option 变化)
    // - 更新 ADSB 失速速度
    // - 写入默认空速到 AHRS
    // - 更新 home 位置
    // - 设置陷波滤波器采样率
}
```
