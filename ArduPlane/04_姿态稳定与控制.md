# 04 — 姿态稳定与控制

> 文件: `ArduPlane/Attitude.cpp`
> 核心: 固定翼 PID 稳定回路 (滚转/俯仰/偏航)

---

## 关键函数

### stabilize() — 主入口 (FAST_TASK)

```cpp
void Plane::stabilize()
{
    // 1. 脚本控制检查 → 如果启用则走脚本路径
    // 2. 否则调用 control_mode->run()
    // 3. 地面零油门时重置积分器
}
```

### stabilize_roll() — 滚转 PID

```
输入: nav_roll_cd (导航需求滚转角度)
反馈: ahrs.roll_sensor (当前滚转角度)
输出: SRV_Channel::k_aileron

特殊处理:
- 倒飞时翻转限幅
- VTOL 模式下从属于多旋翼姿态控制
- 起飞时 tail-hold (尾轮着地禁止滚转)
- 降落 flare 阶段减滚转
```

### stabilize_pitch() — 俯仰 PID

```
输入: nav_pitch_cd (导航需求俯仰角度)
反馈: ahrs.pitch_sensor (当前俯仰角度)
输出: SRV_Channel::k_elevator

特殊处理:
- 倒飞时翻转限幅
- VTOL 模式下从属于多旋翼姿态控制
- 起飞时拉杆 (hold pitch at takeoff angle)
- 降落 flare 强制拉杆
```

### stabilize_yaw() — 偏航控制

三种模式 (根据飞行阶段自动选择):

| 模式 | 触发条件 | 输入/输出 |
|------|---------|----------|
| 航向保持地面转向 | flare + 降落 + 有地面转向 | course_target → steering |
| 角速度地面转向 | 地面 且 非航向保持模式 | rate_target → steering |
| 协调转弯 | 空中 | aileron→rudder 混控 |

### calc_speed_scaler() — 空速 PID 缩放

```
根据当前空速动态缩放 PID 增益。
空速高 → 缩放小 (舵效高，减少输出)
空速低 → 缩放大 (舵效低，增加输出)

使用传感器空速或油门代理，钳位在配置的 min/max 之间。
```

### calc_nav_roll() — 导航滚转限制

```
将导航控制器输出的 roll 需求限制在:
- LIM_ROLL_CD (硬限)
- 载荷因子限幅 (防失速)
```

### calc_nav_pitch() — 导航俯仰限制

```
将导航控制器输出的 pitch 需求限制在:
- LIM_PITCH_MAX/MIN
```

### apply_load_factor_roll_limits() — 载荷因子防失速

```
根据当前空速和载荷因子，限制最大滚转角。
空速低 → 滚转限制小 (防止失速)
空速高 → 滚转限制大
```

---

## 控制回路数据流

```
导航控制器 (TECS/L1)
  → nav_roll_cd    → stabilize_roll()  → aileron PWM
  → nav_pitch_cd   → stabilize_pitch() → elevator PWM
  → throttle        → set_throttle()   → throttle PWM
  → yaw target      → stabilize_yaw()  → rudder PWM / steering PWM

calc_speed_scaler() 影响所有 PID 增益
```
