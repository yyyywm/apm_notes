# 06 — 舵机输出与混控

> 文件: `ArduPlane/servos.cpp`
> 核心: 控制链最后一环，将姿态/导航的输出转换为舵机和电机的 PWM 信号

---

## 关键函数

### set_servos() — 主输出入口 (FAST_TASK)

```
Plane::set_servos() // FAST_TASK，每圈执行
  ├── 1. 输出锁定检查 (cork: 防止地面误动)
  ├── 2. QuadPlane 过渡更新 (如果有)
  ├── 3. set_throttle()     油门计算
  ├── 4. set_servos_flaps()  襟翼混控
  ├── 5. airbrake_update()   气刹更新
  ├── 6. throttle_slew_limit() 油门变化率限制
  ├── 7. 地面/未解锁状态处理
  └── 8. servos_output()    最终混控输出
```

### set_throttle() — 油门计算

```
1. 获取原始油门需求 (来自控制模式)
2. 电池电压补偿
3. MIN/MAX 限制
4. 判定是否需要抑制油门:
   - 未解锁
   - 地面且不在起飞状态
   - 降落伞释放
   - 降落 flare 阶段
   - 自动起飞未达条件
5. THR_MIN 特殊处理 (flare 时保持怠速)
```

### servos_output() — 混控输出

```
1. 双发差速混控 (twin-engine differential)
2. channel_function_mixer():
   - elevon (升降副翼混控) → 一个舵面同时做 roll + pitch
   - vtail (V尾混控) → 两个舵面同时做 pitch + yaw
   - flaperon (襟副翼混控) → 襟翼叠加到副翼
3. Tailsitter / Bicopter 输出
4. 强制 flare (降落拉杆)
5. 差动扰流板
6. 降落后舵面中立
7. SRV_Channels::calc_pwm() → output_ch_all()
```

### throttle_slew_limit() — 油门变化率限制

```
限制每秒油门变化量 (THR_SLEWRATE 参数)
防止油门突变导致飞机不稳
```

### suppress_throttle() — 油门抑制判定

```
返回 true (抑制油门=0) 的条件:
  - 非手动模式 + 地面 + 高度 < 10m + 下降率 < 5m/s
  - 降落伞已释放
  - 降落 flare 阶段
  - GPS 检测到不移动
  - 自动起飞阶段等待条件满足
```

### set_servos_flaps() — 襟翼混控

```
襟翼来源:
  - FLAP_1_SPEED / FLAP_1_PERCNT (空速驱动)
  - 手动通道 (RC 开关)
  - 自动降落阶段
```

### dspoiler_update() — 差动扰流板

```
支持三种 Crow 模式:
  - FLYINGWING: 飞翼
  - FULLSPAN: 全翼展
  - PROGRESSIVE_CROW: 渐进式 Crow

混控源:
  - 副翼信号 → 差动扰流板
  - 方向舵信号 → 扰流板偏航辅助
```

### servos_auto_trim() — 自动微调

```
持续的在线微调:
  1. 监控 roll/pitch I 项 (积分项)
  2. 如果 I 项持续偏大 → 调整 servo trim
  3. 每 10 秒保存到 FRAM

目的: 让 I 项归零，释放积分器容量
```

---

## 混控器类型

| 混控器 | 输入 | 输出 | 适用场景 |
|--------|------|------|---------|
| elevon | roll + pitch | left_aileron, right_aileron | 飞翼/三角翼 |
| vtail | pitch + yaw | left_rudder, right_rudder | V 尾 |
| flaperon | aileron + flap | left_aileron, right_aileron | 襟副翼 |
| dspoiler | aileron + rudder | left_spoiler, right_spoiler | 差动扰流板 |
| twin-engine | throttle + rudder | left_thr, right_thr | 双发差速 |
