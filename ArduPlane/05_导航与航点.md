# 05 — 导航与航点

## 关键函数

### navigate() — 导航主入口 (10Hz)

```
Plane::navigate()  // 10Hz
  ├── 计算 wp_distance / wp_proportion
  ├── 检查 home 高度变化
  ├── 更新 loiter 角度累积
  └── 调用 control_mode->navigate()
```

### update_loiter() — Loiter 导航

```
1. 确定盘旋半径 (WP_LOITER_RAD 或动态计算)
2. 确定盘旋方向 (顺时针/逆时针)
3. 调用 L1_controller.update_loiter()
4. 到达目标后开始计时 (用于 N圈盘旋指令)
```

### calc_airspeed_errors() — 空速误差 (10Hz)

```
目标空速来源 (多源合并):
  - FBWB 模式: 飞手油门杆量映射
  - GUIDED 模式: 外部指令
  - Soaring 模式: 滑翔速度
  - 降落阶段: 着陆速度配置
  - AUTO 模式: 任务指令 (DO_CHANGE_SPEED)

最终叠加:
  实际目标 = 基础目标 + 空速微调 (nudge) + 地速补偿
  误差 = 目标空速 - 实际空速 → 送入 TECS
```

### calc_gndspeed_undershoot() — 地速补偿

```
如果地速 < MIN_GROUNDSPEED:
  增加空速目标，确保飞机在地面坐标系中有足够前进速度
```

### update_fbwb_speed_height() — FBWB/CRUISE 高度控制

```
FBWB/CRUISE 的独特控制策略:
  - 升降舵 → 控制高度 (调整升降影响爬升率)
  - 油门 → 控制空速 (调整油门影响前进速度)
```

## commands.cpp — 航点管理

| 函数 | 作用 |
|------|------|
| `set_next_WP(loc)` | 加载下一个航点，设置 crosstrack / alt_slope / turn_angle |
| `set_guided_WP(loc)` | 设置 Guided 目标，禁用 crosstrack |
| `update_home()` | 自动从 GPS 更新 home 位置 |

### set_next_WP() 流程

```
1. 保存 pre_WP_loc = next_WP_loc
2. 设置 next_WP_loc = 新位置
3. 如果 lat/lng = 0 → 原地 loiter
4. 地形修正
5. 检查是否已飞过航点
6. setup_alt_slope() — 设置高度变化斜率
7. setup_turn_angle() — 设置转弯
8. 重置 loiter 角度和 crosstrack
```

## 导航控制器 (AP_L1_Control)

L1 算法: 飞机跟踪 L1 距离前方的参考点来产生横向加速度指令。

```
wp_distance → L1 距离 (根据空速动态调整)
bearing_error → 横向加速度 → roll 指令
```
