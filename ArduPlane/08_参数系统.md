# 08 — 参数系统

> 文件: `ArduPlane/Parameters.h` + `Parameters.cpp`
> 核心: 参数定义、索引、存储和访问

---

## 参数分组

ArduPlane 有两组参数 (因为单组最多 256 个索引):

- **Parameters** (class `g`) — 第一组，约 70 个参数
- **ParametersG2** (class `g2`) — 第二组，存放新增参数和子对象

全局访问:
```cpp
g.param_name    // 第一组
g2.param_name   // 第二组
```

---

## 参数定义格式 (在 Parameters.cpp 中)

```cpp
// @Param: ENABLE        ← 参数名 (最多 16 字符)
// @DisplayName: Terrain enable
// @Description: enable terrain data
// @Values: 0:Disable,1:Enable
// @Range: 0 1
// @User: Advanced
AP_GROUPINFO_FLAGS("ENABLE",    // 参数名
    0,                           // 索引 (绝对不能改!)
    AP_Terrain,                  // 所属类
    enable,                      // 成员变量名
    1,                           // 默认值
    AP_PARAM_FLAG_ENABLE),       // 参数标志
```

**关键规则: 永远不要修改已有参数的索引值。** 新增参数只能追加在末尾。

## 主要参数分类

### 姿态控制类
- `RLL2SRV_*` — 滚转 PID
- `PTCH2SRV_*` — 俯仰 PID
- `YAW2SRV_*` — 偏航 PID
- `KFF_RDDRMIX` — 方向舵-副翼混控
- `KFF_THR2PTCH` — 油门-俯仰前馈
- `STICK_MIXING` — 飞手杆量混控

### 导航类
- `WP_LOITER_RAD` — 盘旋半径
- `WP_RADIUS` — 航点到达半径
- `ALT_SLOPE_MIN/MAX` — 高度变化斜率

### 失控类
- `FS_ACTION_SHORT/LONG` — 失控动作
- `FS_LONG_TIMEOUT` — 长时失控触发时间
- `BATT_FS_ACTION` — 电池失控动作

### 起飞/降落类
- `TKOFF_*` — 起飞参数
- `LAND_*` — 降落参数

### QuadPlane 类
- `Q_ENABLE`, `Q_FRAME_CLASS/TYPE`, `Q_TRANSITION_MS`, `Q_LAND_*`, `Q_ASSIST_*`

## 参数注解

| 注解 | 说明 |
|------|------|
| `@Param` | 参数名 |
| `@DisplayName` | UI 显示名 |
| `@Description` | 参数说明 |
| `@Values` | 可选值 (0:Disable,1:Enable) |
| `@Bitmask` | 位掩码 (每个 bit 的含义) |
| `@Range` | 取值范围 |
| `@Units` | 单位 |
| `@Increment` | UI 步进值 |
| `@User` | 用户级别 (Standard/Advanced) |
| `@RebootRequired` | 需要重启生效 |
