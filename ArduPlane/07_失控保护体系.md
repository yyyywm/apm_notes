# 07 — 失控保护体系

> 文件: `ArduPlane/events.cpp` + `radio.cpp` + `defines.h`
> 核心: RC 失控、GCS 心跳丢失、电池低压的检测与响应动作

---

## 失控状态枚举 (defines.h)

```cpp
FAILSAFE_NONE  = 0  // 正常
FAILSAFE_SHORT = 1  // 短时失控 (RC 信号短暂丢失)
FAILSAFE_LONG  = 2  // 长时失控 (RC 或 GCS 长时间无信号)
FAILSAFE_GCS   = 3  // GCS 失控
```

---

## RC 失控处理

### 短时失控 (radio.cpp → events.cpp)

```
检测: read_radio() 每 50Hz 检查 RC 有效位
触发: 连续 N 次无效 → FAILSAFE_SHORT

failsafe_short_on_event():
  根据 FS_ACTION_SHORT 参数:
    - 0: BESTGUESS (CIRCLE/不变)
    - 1: CIRCLE
    - 2: FBWA
    - 3: DISABLED
    - 4: FBWB
  VTOL 模式特殊处理: QLAND 替代

failsafe_short_off_event():
  恢复模式: 如果飞手没有手动切模式 → 回到之前保存的模式
```

### 长时失控

```
检测: check_long_failsafe() 每 3Hz
触发条件:
  - RC 短时失控持续时间 > FS_LONG_TIMEOUT
  - 或 GCS 心跳丢失 > FS_GCS_TIMEOUT

failsafe_long_on_event():
  根据 FS_ACTION_LONG 参数:
    - 0: CONTINUE (不动作)
    - 1: RTL
    - 2: GLIDE (滑翔)
    - 3: PARACHUTE (降落伞)
    - 4: AUTO (继续任务)
    - 5: AUTOLAND (自主降落)

failsafe_long_off_event():
  清除状态，恢复模式
```

---

## 电池失控

```
handle_battery_failsafe():
  根据 BATT_FS_ACTION 参数:
    - 0: 无动作
    - 1: QLand (VTOL 降落)
    - 2: Loiter + QLand 备降
    - 3: Land (跳到降落航点)
    - 4: RTL
    - 5: Autoland 或 RTL
    - 6: Terminate (立即停机)
    - 7: Parachute (降落伞)
```

---

## 失控状态机总览

```
正常状态
  │
  ├── RC 信号短暂丢失 → FAILSAFE_SHORT
  │   ├── 恢复 → 回到正常
  │   └── 超时 (FS_LONG_TIMEOUT) → FAILSAFE_LONG
  │       └── 动作: RTL/GLIDE/PARACHUTE/AUTO/AUTOLAND
  │
  ├── GCS 心跳超时 (FS_GCS_TIMEOUT) → FAILSAFE_GCS
  │   └── 动作: 同 FAILSAFE_LONG
  │
  └── 电池低压 → BATT_FS
      └── 动作: QLand/Loiter/Land/RTL/Terminate/Parachute
```

---

## 相关参数

| 参数 | 说明 |
|------|------|
| `FS_ACTION_SHORT` | 短时失控动作 |
| `FS_ACTION_LONG` | 长时失控动作 |
| `FS_LONG_TIMEOUT` | 长时失控触发时间 (秒) |
| `FS_GCS_TIMEOUT` | GCS 失控触发时间 (秒) |
| `FS_GCS_ENABL` | GCS 失控使能 |
| `BATT_FS_ACTION` | 电池失控动作 |
| `THR_FS_VALUE` | 油门失控检测值 |
| `THR_FS_ACTION` | 油门失控动作 |
